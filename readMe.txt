1- Nella certella ROS ci sono tutti i file necessari per avviare un'esplorazione con ROS, ed ottenere tutte le mappe parziali relative ad una mappa.
2- Nella cartella Input ci sono tutte le mappe usate come input per il sistema
3- Nella cartella Output ci sono tutti gli output del sistema 
4- Nella cartella refactored c'è il progetto 
5- Nella cartella conversione mappe parzieli  c'è il progetto che permette di convertire tutte le mappe parziali in formato .png.
6- Nella cartella tesi c'è la versione finale della tesi di Matteo.
7- Nella cartella legacy ci sono vecchie informazioni
8- Nella cartella dataset mappe parzieli, ci sono tutte le mappe parziali provenienti dall'esplorazione con ROS, e che vengono convertiti in formato .png grazie a conversione_mappe_parziali (punto 5).
